# Cyber Portfolio

A static terminal-inspired cybersecurity portfolio for Mohammed Ayaz Ansari, based on the style of `serozr/cyber-portfolio`.

## Customize

Edit `index.html` to update:

- your LinkedIn URL
- any GitHub or lab write-up links
- project details and screenshots
- timeline items
- skill names and levels

## Run

Open `index.html` directly in a browser, or run a local server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
